# Spanish TIE cita check

This is a set of tests for automated check of cita availabitily for TIE application in Valencia, Spain.

## How to run check locally

CitaCheckTest.java contains several tests that will check cita availability for all possible immigration offices in this province.

Please adjust values of NIE, Name and Nationality before executing tests.

## Configuration for another Spanish province

START_URL value is set for Valencia province, as well as office addresses inside Office.enum. Please adjust values for your location if it's needed
